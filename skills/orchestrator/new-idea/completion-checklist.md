# Completion Checklist: New Idea

Before using `summary.response.md`, confirm:

- [ ] Page idea is stated in one sentence.
- [ ] Page type is provided or explicitly marked unknown.
- [ ] Purpose is clear.
- [ ] Target audience is clear.
- [ ] Desired visitor outcome is clear.
- [ ] Why the page matters is captured.
- [ ] References or style input are captured, or marked none.
- [ ] Avoid list is captured, or marked none.
- [ ] Success definition is captured.
- [ ] Assumptions are clearly labeled.
- [ ] Summary asks the human to approve, modify, or reject.
