# Missing Input Response: New Idea

Use this response when the human's idea is not clear enough to create a kickoff brief.

Ask only for the missing items:

- What webpage do you want to create?
- What type of page is it, if you know? For example: landing page, SaaS page, portfolio, product page, waitlist, documentation, pricing, event page.
- Who is the page for?
- What should visitors do after reading it?
- Why does this page matter?
- Are there any references, styles, or competitors to consider?
- What should the page avoid?
- How will you know the page worked?

Keep the response concise and do not continue to discovery yet.
