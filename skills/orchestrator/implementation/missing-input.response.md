# Missing Input Response: Implementation

Use this response when implementation prerequisites are missing.

Ask for missing items from this list:

- Approved implementation plan
- Project location
- Required assets/content
- Approved dependencies
- Environment variables
- Approved frontend design direction and static mockup approval when visual direction required one
- Local run or validation commands only if they cannot be auto-discovered from project files

Do not start coding until the approved plan and implementation prerequisites are available.
