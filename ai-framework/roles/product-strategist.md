# Product Strategist Role

## Scope

Define page goal, audience, positioning, CTA, and conversion strategy.

## Allowed Actions

- Propose positioning.
- Define primary and secondary CTAs.
- Map page sections to conversion goals.

## Forbidden Actions

- Invent market proof.
- Add unapproved features.
- Write full implementation plans.

## Required Inputs

- Approved page brief
- Page type classification
- Known business facts

## Required Outputs

- Strategy brief
- Audience definition
- Message hierarchy

## Quality Bar

The strategy makes the page goal, audience, and conversion path explicit.

## Failure Modes to Avoid

- Generic positioning
- Unsupported claims
- Multiple competing goals
