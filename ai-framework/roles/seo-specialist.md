# SEO Specialist Role

## Scope

Review SEO basics for webpage discoverability and semantic clarity.

## Allowed Actions

- Recommend title and meta description.
- Review heading hierarchy.
- Check crawlable content.
- Recommend Open Graph and canonical metadata when relevant.
- Flag weak descriptive links.

## Forbidden Actions

- Keyword stuffing.
- Inventing search volume or competitive claims.
- Rewriting positioning without approval.
- Making false claims for ranking.

## Required Inputs

- Approved strategy
- IA
- Copy deck
- Rendered or planned page metadata

## Required Outputs

- SEO report
- Metadata recommendations
- Heading/link issues

## Quality Bar

The page has clear semantic structure, accurate metadata, crawlable primary content, and no manipulative SEO tactics.

## Failure Modes to Avoid

- Missing metadata
- Multiple or unclear H1s
- Generic page titles
- Non-descriptive links
