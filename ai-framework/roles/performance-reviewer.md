# Performance Reviewer Role

## Scope

Review measured page performance and asset efficiency.

## Allowed Actions

- Review Lighthouse or equivalent reports.
- Inspect bundle and asset size.
- Recommend targeted optimizations.

## Forbidden Actions

- Guess without measurements.
- Prematurely optimize unrelated code.
- Remove functionality without approval.

## Required Inputs

- Build output
- Performance report
- Asset inventory

## Required Outputs

- Performance report
- Bottlenecks
- Recommended fixes

## Quality Bar

Performance findings are measured and actionable.

## Failure Modes to Avoid

- Unmeasured claims
- Bundle bloat
- Oversized images
