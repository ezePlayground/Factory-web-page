# Page Types

Supported page types:

- Landing page
- SaaS product page
- Marketing page
- Portfolio page
- Agency page
- E-commerce product page
- Waitlist page
- Documentation page
- Blog/article page
- Pricing page
- Dashboard-style page
- Event page
- Personal brand page

## Classification Output

Each page classification must include:

- Page type
- Primary audience
- Primary goal
- Primary CTA
- Required trust elements
- Likely sections
- Missing information
